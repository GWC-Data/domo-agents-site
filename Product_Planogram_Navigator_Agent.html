Agent Overview

The Product Planogram Navigator Agent intelligently restructures product placement by analyzing historical POS data, customer purchase journeys, and product affinities. It dynamically suggests shelf arrangements that maximize visibility and efficiency while considering spatial and inventory constraints. It acts as an automated merchandising strategist for real-time planogram adjustments.

Problem Statement

While traditional analytics reveal product affinities, they lack spatial reasoning and adaptability. ML models can suggest bundles but do not account for shelf space or category overlaps. Manual merchandising layouts are often static, slow to update, and don’t reflect real-time sales patterns or inventory limitations.

Why Not Just BI / ML / Workflow?

BI tools visualize affinity but don't act on it. ML models are limited to association rules without spatial awareness. Static workflows can’t adapt to fast-moving inventory or demand shifts. This agent provides context-aware planogram logic, evaluates shelf efficiency, and triggers merchandising updates automatically based on evolving data.

Agent Capabilities

✅ Analyzes purchase sequence and affinity groups
✅ Evaluates product visibility and performance shelf-wise
✅ Detects shelf underperformance or overcrowding
✅ Recommends staff shifts between zones to optimize coverage
✅ Recommends optimal shelf placement by category, affinity, and velocity
✅ Supports automated planogram refresh via API

Workflow Explanation

Sales Intelligence Catalyst:
  • Reads historical POS data and product attributes
Shelf Efficiency Evaluator Catalyst:
  • Assesses current product placement shelf-by-shelf for efficiency and movement
Planogram Recommendation Catalyst
  • Evaluates products by shelf
  • Recommends new shelf placements based on visibility, affinity scores, and performance
End
  • Updated planogram is generated and can be pushed via merchandising system APIs

Tools & Components Used

* Domo Workflows
* MagicETL (to process POS and shelf allocation data)
* Agent Prompts (for shelf logic and affinity scoring)
* Custom Brick (for planogram visualization)
* Planogram API Integrations (for dynamic updates)
